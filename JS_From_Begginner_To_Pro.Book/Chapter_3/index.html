Array methods

Adding and replacing

  1. push() (adds new element into array)

    example:array.push("element")

  2. splice() - adds elements at certain index.

    example:array.splice(2, 0, "element") - element will be added from second index of the array and 0 means we won't delete 0 elements after second index we just added, so the elements after addition will shift positions ascending wise

  3. concat() - adds 1 array into another array.

    example:array.concat(array2) or let array3 = array.concat(array2)

Deleting elements

  1. pop() - Deletes the last element from the array

    example:array.pop()

  2. shift() - Deletes the first element from the array

    example:array.shift()

  3. splice() - Also Deletes element from the array

    example: array.splice(1,5)
